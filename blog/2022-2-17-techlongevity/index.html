The Longevity of Old Computers and Software

First published Feb

Old browsers thrive on the dark web

There are a good number of websites that would still be quite viewable with older web browsers, but nearly every clearnet site these days uses HTTPS, making them completely unreachable with old browsers that don't support modern encryption.

However, websites on anonymity networks like Tor and I2P don't need HTTPS since the nature of the network already handles all the encryption and authenticity. Therefore, (almost) all sites on the dark web use plain HTTP, making them accessible with any old browser configured to use a darkweb proxy.

A lot of the web design I see on the dark web is often simpler than all the bloated sites on the clearnet, so they often work really well with older browsers.
