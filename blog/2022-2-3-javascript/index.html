Old Man Yells at JavaScript

First published Feb 3, 2022 at 3:59 AM

The modern web is a bloated mess, and JavaScript has played a huge role in that, which has prompted me to create a website dedicated to not using JavaScript:

https://disable-javascript.com

True to its name, this website shows how to disable JavaScript in popular browsers or using the tracker and script blocker uBlock Origin. There are also pages explaining how JS is a bad language, is annoying, comprimises security, and invades privacy.

The site avoids using any JS whatsoever. Not even the "JS is currently enabled/disabled" message uses JavaScript thanks to clever use of the <noscript> tag and radio buttons.

To help other website owners use less JavaScript, there is a dynamic CSS page showing how to use CSS for things commonly done with JS. If there is more of this CSS on most websites, there should be less of a reason to have to enable JS on those sites.

At the time of writing this, the site's content feels a bit too negative, so I will work on wording it differently. Or do I have anxiety?
